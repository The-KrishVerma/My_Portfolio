## Features

*   **Responsive Design:** The portfolio is optimized for both desktop and mobile devices.
*   **Interactive UI:** The portfolio includes various interactive elements, such as animations and hover effects.
*   **Contact Form:** A functional contact form that allows users to send messages to me.
*   **Project Showcase:** A section to display my projects.
*   **Skills and Experience:** A detailed overview of my skills and experience.
*   **Coding Profiles:** Links to my profiles on various competitive programming platforms.
