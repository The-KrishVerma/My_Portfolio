**Live Site:** [Live Link](https://the-krishverma.vercel.app/)

## About Me

I am a passionate and driven B.Tech IT student at IIIT Allahabad with a strong desire to solve real-world problems through technology. I am enthusiastic about web development, mobile applications, and other emerging technologies.

## Features

*   **Dynamic & Responsive Design:** The portfolio is fully responsive and built with a mobile-first approach, ensuring a seamless experience across all devices.
*   **Interactive UI:** The user interface is designed to be engaging and interactive, with smooth animations and transitions.
*   **Project Showcase:** A dedicated section to display my projects, complete with descriptions, technologies used, and links to live demos and source code.
*   **Skills & Experience:** Detailed sections outlining my technical skills and professional experience.
*   **Contact Form:** An integrated contact form for easy communication.

## Tech Stack

This portfolio is built with the following technologies:

*   **Frontend:** React, TypeScript, Tailwind CSS, Shadcn UI
*   **Build Tool:** Vite
*   **Deployment:** Netlify
